# Bonus Lab

Use everything everything you have learned so far in this course to create the magic conch script I used the first few weeks of classes. Take in a list of names (Firstname Lastname) from a file

Requirements:
1. Write the output using art.tprint()
2. Take user input for the number of names to display in the output
3. The magic conch should read from a list of names in a text file, then select a random number of names equal to the user input.
4.  At least 100 shell emojis are displayed in the output
